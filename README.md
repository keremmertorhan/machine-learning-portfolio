# Machine Learning Portfolio

Welcome to my Machine Learning Portfolio! This repository contains a collection of projects demonstrating my understanding of fundamental and advanced ML algorithms, spanning across Regression, Classification, Clustering, and Computer Vision.

## Projects Overview

### 1. [Life Expectancy Analysis (Linear Regression)](./Linear_Regression_LifeExpectancy)
* **Goal:** Predicting life expectancy based on socio-economic factors (GDP, health expenditure, etc.).
* **Techniques:** Exploratory Data Analysis (EDA), Simple Linear Regression, R² Score, MAE.

### 2. [Iris Classification (Decision Trees)](./DecisionTree_Iris)
* **Goal:** Classifying Iris flowers into species based on physical dimensions.
* **Techniques:** DecisionTreeClassifier, Train/Test splitting, Overfitting analysis, Stability testing across random states.

### 3. [Coordinate & Spiral Clustering (K-Means)](./KMeans_Clustering)
* **Goal:** Grouping unlabeled 2D datasets and analyzing algorithm limitations on complex shapes (like spirals).
* **Techniques:** Unsupervised Learning, K-Means Clustering, Centroid visualization.

### 4. [MNIST Digit Recognition (Naive Bayes)](./NaiveBayes_MNIST_ImageClassification)
* **Goal:** Classifying handwritten digits (0-9) using pixel data.
* **Techniques:** Multinomial Naive Bayes, Confusion Matrix, Error rate analysis per class.

## 🛠️ Technologies & Libraries Used
* **Language:** Python
* **Libraries:** `scikit-learn`, `pandas`, `numpy`, `matplotlib`
